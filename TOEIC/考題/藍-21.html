解題關鍵
Whoever 本身已含先行詞（= anyone who），可直接作主詞引導名詞子句，後接動詞 wish，結構完整且正式，最符合本句商業通知語境。

文法補充
Whoever = anyone who，可直接作主詞或受詞，本身已包含先行詞，無需另加 who。正式書面語（公告、備忘錄）中常以 Whoever / Whomever 起首。

Q7
No sooner had the new lease been approved ＿＿ the facilities team began preparing the unit for occupancy.
【中譯】新租約一獲批准，設施團隊便立刻開始準備該單元供入住使用。
A
than
B
when
C
that
D
as
解題關鍵
「No sooner had + S + p.p. + than + S + V（過去式）」是表示「一…就…」時間緊接關係的否定倒裝句型，主要子句用過去完成式，than 引導的後半子句用一般過去式。than 是此句型唯一正確搭配詞。
陷阱提示
when 是最常見陷阱——Hardly/Scarcely...when（或 before）可用 when，但 No sooner 固定搭配 than，考試中兩組句型的連接詞不可互換。
文法補充
No sooner 否定倒裝：No sooner had + S + p.p. ... than + S + V（past）。否定副詞 no sooner 置於句首時，主句主謂倒裝（had + S + p.p.）；than 引導結果/後續子句，動詞用一般過去式。比較：Hardly/Scarcely + had + S + p.p. ... when/before + S + V。
單字/片語詞性音標主要意思搭配用法
thanconj.—（No sooner...than 固定搭配連接詞）No sooner had + S + p.p. + than + S + V（過去式）✓
whenconj.—（時間連接詞）Hardly/Scarcely...when 可用 when，但 No sooner 固定搭配 than，不可換成 when
thatconj.—（名詞子句或結果副詞子句連接詞）that 不引導 no sooner 倒裝結構中的後半子句
asconj.—（當…時；隨著）as 雖可表時間但不用於 no sooner 倒裝固定句型
