✦ 科技合規系統更新
Q1
The software update will not be deployed across the corporate network ＿＿ all security patches have been verified and formally approved by the IT governance committee.
【中譯】該軟體更新不會在企業網路上部署，__所有安全修補程式均經過驗證並獲得 IT 治理委員會的正式批准。
A
unless
B
until
C
whether
D
although
解題關鍵
unless + S + V = 條件否定：「若非…，則不…」；will not be deployed unless all patches are verified = 只有在所有修補程式通過驗證的情況下才部署
陷阱提示
until 與 unless 在語意上相近但層面不同：until 強調時間截止點（等到某時），unless 強調條件否定（若非某條件成立）；本題重點在條件成立與否，not before 的時間語意不如 unless 的條件語意精準
文法補充
unless vs. until vs. if not 辨析：① unless + S + V = if + S + not + V（條件否定，語意等效）② until + S + V = up to the point when S + V（時間截止點）③ 本題：will not be deployed unless all patches are verified ≈ will not be deployed if all patches have not been verified ④ 注意：until 有時語意接近 unless（會被混淆），但 unless 更強調「條件不成立則不發生」的邏輯關係
單字/片語詞性音標主要意思搭配用法
unlessconj./ʌnˈlɛs/除非；若非（條件否定連接詞）will not + V unless + S + V = 「唯有條件成立時才執行，否則不執行」✓（條件否定：absence of condition → absence of action）
untilconj./ʌnˈtɪl/直到（時間截止點連接詞）will not + V until + S + V = 「直到某時點才發生」（強調時間上的截止等待；此處語意重點在條件是否滿足，而非時間截止，unless 更精準）
whetherconj./ˈweðər/是否（名詞子句引導詞）whether + S + V（引導名詞子句表「是否」，不作條件否定連接詞；語法結構不符）✗
althoughconj./ɔːlˈðoʊ/雖然（讓步連接詞）although + S + V（讓步語氣，表「雖然如此，主句仍成立」；與本題條件否定語意完全相反）✗
Q2
------- branch manager is required to submit a monthly performance report to headquarters by the fifth of each month.
【中譯】每一位分行經理都必須在每月五日前向總部提交月度績效報告。
A
All
B
Both
C
Each
D
Every
解題關鍵
空格後 branch manager 為單數名詞，謂語動詞為 is required（單數）。Each 與 Every 均接單數可數名詞，但 Each 可置於句首後接單數名詞+單數動詞，且語意強調「逐一、個別地」，最符合此句強調個別責任的語境。All 與 Both 需接複數名詞（All managers / Both managers），與空格後 manager（無 -s）不符，可排除。Every 雖也接單數名詞，但通常不用於句首強調個別義務，Each 更為正式且精確。
陷阱提示
Every manager 語法上亦可行，但考題設計中空格後為 branch manager（單數），All 和 Both 即被排除；Each 比 Every 更強調個體責任分配，在正式商業文書中為首選。
文法補充
each/every + 單數可數名詞 + 單數動詞；all/both + 複數可數名詞 + 複數動詞。Each 強調逐一，Every 強調全部中無一例外，語意相近但搭配場合有別。
單字/片語詞性音標主要意思搭配用法
All限定詞/ɔːl/所有的（複數）All managers are
Both限定詞/boʊθ/兩者都（複數）Both managers are
Each限定詞/iːtʃ/每一個（單數）Each manager is
Every限定詞/ˈɛvri/每一個（單數）Every manager is
Q3
It is a common error to ＿＿ correlation with causation — the fact that two variables move together does not necessarily mean that one causes the other.
【中譯】將相關性與因果關係__是一個常見錯誤——兩個變數同向移動並不必然意味著其中一個導致另一個。
A
conflate
B
confuse
C
contrast
D
combine
解題關鍵
conflate A with B = C1 正式「將A與B不當地混為一談/等同」；confuse 偏口語混淆；contrast 語意相反；combine 中性不含「錯誤」義
陷阱提示
confuse 與 conflate 均有「混淆」義，但 confuse 偏「感到迷惑或使人混亂」，conflate 更正式且特指「將兩個不同概念錯誤地視為同一」，學術論述用 conflate 更精準
單字/片語詞性音標主要意思搭配用法
conflatev/kənˈfleɪt/混為一談；不當合併（C1）conflate correlation with causation ✓（C1正式：將兩個不同概念錯誤地視為同一）
confusev/kənˈfjuːz/混淆；使困惑confuse the two（語意相近但語域偏口語；不如 conflate 精確描述「不當合併」）
contrastv/ˈkɒntrɑːst/對比；形成對照contrast two ideas（語意相反：對比區分，非混為一談）
combinev/kəmˈbaɪn/結合；合併combine ingredients（中性結合，不含「錯誤等同」的負面語意）
字詞家族

解題關鍵
Scarcely had + S + 過去分詞（p.p.）+ when + 過去式主句 = 否定副詞前置倒裝完成式，表「幾乎還沒……就……」；had 後直接接過去分詞。

文法補充
