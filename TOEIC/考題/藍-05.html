Q5
＿＿ the supplier notified us of the delay earlier, we could have adjusted the production schedule accordingly.
【中譯】假如供應商早些通知我們延誤的情況，我們本可以相應地調整生產時程。
A
Had
B
If
C
Should
D
Were
解題關鍵
'Had + S + p.p.' = 省略 if 的過去假設倒裝句，等同 'If S had + p.p.'；後接 would/could have + p.p.
陷阱提示
'Should' 對應現在/未來假設；'Were' 對應現在假設；此處為過去未發生的假設，須用 Had
文法補充
省略 if 的倒裝假設句：過去→ Had S p.p., S would/could have p.p.；現在/未來→ Were S to V / Should S V；此結構常見於正式書面英語
單字/片語詞性音標主要意思搭配用法
Hadaux.（省略 if 的倒裝假設；過去完成倒裝）Had + S + p.p. = If S had + p.p. ✓
Ifconj.（標準條件句引導詞）文法正確，但此題測試省略 if 的倒裝形式
Shouldaux.（省略 if 的現在/未來假設倒裝）Should + S + V = If S should V（現在/未來假設，非過去）
Wereaux.（be 動詞倒裝假設；現在假設）Were + S + to V（現在/未來假設）；過去假設用 had

Q11
The committee has not yet decided ＿＿____ for the upcoming fiscal year.
【中譯】委員會尚未決定______下一個財政年度。
A
whether the proposal will be approved
B
if the proposal will be approved
C
whether the proposal would approved
D
that whether the proposal is approved
解題關鍵
whether the proposal will be approved
陷阱提示
B 的 if 雖可引導名詞子句，但在正式書面語中 whether 更適當，且 if 子句一般不以 will 表達未來意圖（易與條件句混淆）；C 缺少 be 動詞；D 重複使用引導詞。
文法補充
whether vs if 用於名詞子句：(1) whether 為正式用法，可置於句首、介系詞後，以及表示不確定的未來事件；(2) if 偏口語，不可置於句首或介系詞後（如 decided if 在正式文件中不佳）；(3) 此句中 decided 後接名詞子句作賓語（Noun Clause as Object），whether 為正確選擇。
單字/片語詞性音標主要意思搭配用法
whether the proposal will be approved名詞子句（whether 引導，正式語境正確用法）✓ whether 可用於介系詞後及正式書面語，接 will 表真實未來
if the proposal will be approved名詞子句（if 引導，口語用法）if 引導名詞子句偏口語，且一般不與 will 並用於此結構
whether the proposal would approved缺少 be，語法不完整would approved 缺少助動詞 be，應為 would be approved
that whether the proposal is approvedthat 與 whether 重複，結構錯誤引導詞重複（that + whether），為常見錯誤結構

解題關鍵
「a number of + 複數名詞」作主詞時，動詞用複數（have）。相反，「the number of + 複數名詞」作主詞時，動詞用單數（is/was）。本句主詞為 a number of employees，故選 have。
陷阱提示

文法補充
a number of + 複數名詞 → 複數動詞；the number of + 複數名詞 → 單數動詞
