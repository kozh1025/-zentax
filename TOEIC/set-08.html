✦ 科技合規系統更新
Q1
The software update will not be deployed across the corporate network ＿＿ all security patches have been verified and formally approved by the IT governance committee.
【中譯】該軟體更新不會在企業網路上部署，__所有安全修補程式均經過驗證並獲得 IT 治理委員會的正式批准。
A
unless
B
until
C
whether
D
although
解題關鍵
unless + S + V = 條件否定：「若非…，則不…」；will not be deployed unless all patches are verified = 只有在所有修補程式通過驗證的情況下才部署
陷阱提示
until 與 unless 在語意上相近但層面不同：until 強調時間截止點（等到某時），unless 強調條件否定（若非某條件成立）；本題重點在條件成立與否，not before 的時間語意不如 unless 的條件語意精準
文法補充
unless vs. until vs. if not 辨析：① unless + S + V = if + S + not + V（條件否定，語意等效）② until + S + V = up to the point when S + V（時間截止點）③ 本題：will not be deployed unless all patches are verified ≈ will not be deployed if all patches have not been verified ④ 注意：until 有時語意接近 unless（會被混淆），但 unless 更強調「條件不成立則不發生」的邏輯關係
單字/片語詞性音標主要意思搭配用法
unlessconj./ʌnˈlɛs/除非；若非（條件否定連接詞）will not + V unless + S + V = 「唯有條件成立時才執行，否則不執行」✓（條件否定：absence of condition → absence of action）
untilconj./ʌnˈtɪl/直到（時間截止點連接詞）will not + V until + S + V = 「直到某時點才發生」（強調時間上的截止等待；此處語意重點在條件是否滿足，而非時間截止，unless 更精準）
whetherconj./ˈweðər/是否（名詞子句引導詞）whether + S + V（引導名詞子句表「是否」，不作條件否定連接詞；語法結構不符）✗
althoughconj./ɔːlˈðoʊ/雖然（讓步連接詞）although + S + V（讓步語氣，表「雖然如此，主句仍成立」；與本題條件否定語意完全相反）✗

【中譯】品質保證團隊將在批准新製造流程全面投產之前，__一系列產品測試。

Q3
Visitors should proceed to the registration desk, ＿＿ staff members will issue name badges and distribute event programs.
【中譯】訪客應前往報到台，__工作人員將發放姓名牌並分發活動手冊。
A
where
B
which
C
that
D
when
解題關鍵
registration desk 是地點名詞，後接的關係子句用關係副詞 where 引導，等同 at which（at the registration desk）。子句中已有完整主謂（staff members will issue...），故不用關係代名詞 which/that。
陷阱提示
which 是最常見陷阱——which 作關係代名詞時需在子句中充當主詞或受詞，但此處子句主詞已是 staff members，which 若插入則造成主詞重複；應改用充當地點狀語的 where。
文法補充
關係副詞 where：先行詞為地點名詞（place / office / area / desk / room）時，關係子句用 where 引導，where = at/in which。結構：先行詞（地點）+ where + S + V，子句內不需另一個主詞位置。
單字/片語詞性音標主要意思搭配用法
whererel.adv.—（關係副詞，修飾地點名詞）registration desk = 地點先行詞 → where 引導非限制性關係子句 ✓
whichrel.pron.—（關係代名詞）which 代替先行詞作子句主詞或受詞，但此處子句已有主詞 staff members，不需再用 which
thatrel.pron.—（關係代名詞）that 不能引導非限制性（補充說明型）關係子句，且不作關係副詞
whenrel.adv.—（關係副詞，修飾時間名詞）when 修飾表時間的先行詞，不修飾表地點的 desk

Q6
The newly enacted Regulation 2026/14 will ＿＿ all previous data handling guidelines, requiring companies to update their internal protocols within sixty days of publication.
【中譯】新頒布的2026/14號規定將__所有先前的資料處理準則，要求各公司在發布後六十天內更新內部規程。
A
supersede
B
replace
C
revoke
D
override
解題關鍵
supersede = 正式法律/政策動詞，指新版本或新規定廢除並全面取代原有者；supersede existing guidelines/regulations 為標準搭配，常見於法規更新或版本迭代語境
陷阱提示
replace 語域較通用，法律文件少用；revoke 偏撤銷已授予的特定權利或授權；override 偏以更高權威凌駕，均不如 supersede 精準描述新規取代舊規的法律語境
單字/片語詞性音標主要意思搭配用法
supersedev/ˌsuːpərˈsiːd/取代；廢除（以新規則或版本替換舊有者）supersede existing rules/regulations/policies ✓（正式法律書面動詞，表新版本系統性地取代並廢除舊有規定）
replacev/rɪˈpleɪs/替換；取代（通用詞，偏物品或人員替換）replace a part/person（通用替換詞，語域偏口語，法律政策文件中不如 supersede 正式精準）
revokev/rɪˈvoʊk/撤銷；廢除（強制取消授權或已授予的權利）revoke a license/permit/authorization（偏「強制撤銷授權或特定許可」，不描述新規定系統性取代舊準則的漸進性過渡）
overridev/ˌoʊvərˈraɪd/凌駕；否決（以更高權威壓制）override a decision/veto（偏「以更高權威否決或壓制」，不描述新規定全面系統性地取代舊規）
字詞家族

Q12
The board of directors has proposed that every department head ＿＿ a quarterly performance review by the end of next month.
【中譯】董事會已提議，每位部門主管 __ 於下個月底前完成季度績效審查。
A
submits
B
submit
C
submitted
D
to submit
解題關鍵
propose 屬於「建議/要求類動詞」（suggest, recommend, demand, require, insist 等），其後接 that 子句時，子句動詞須用「原形動詞」（假設性虛擬語氣），無論主詞人稱或數為何。
陷阱提示
不可因主詞 every department head 為第三人稱單數而選 submits；虛擬語氣 that 子句中動詞形式固定為原形，不隨主詞變化。to submit 是不定詞形式，不用於此結構。
文法補充
建議/要求類動詞（propose, suggest, recommend, demand, request, require, insist）後接 that 子句時，子句動詞使用原形動詞（假設性虛擬語氣 / subjunctive mood），例：It is essential that he submit（非 submits）the report.
單字/片語詞性音標主要意思搭配用法
submitsv. (third-person singular present)/səbˈmɪts/提交（第三人稱單數現在式）一般陳述句的第三人稱單數變化
submitv. (base form)/səbˈmɪt/提交（原形動詞）虛擬語氣中 that 子句使用原形動詞
submittedv. (past tense)/səbˈmɪtɪd/提交（過去式）過去發生的動作
to submitinfinitive/tə səbˈmɪt/提交（不定詞）常用於 ask/want sb to do，不用於虛擬語氣 that 子句
