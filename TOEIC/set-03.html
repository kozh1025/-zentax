Q1
According to the latest HR study, ＿＿ employees attend professional training, __ they tend to be in their daily tasks.
【中譯】根據最新人力資源研究，員工參加專業培訓的頻率__高，日常工作中的生產力__強。
A
the more regularly / the more productive
B
the more regularly / more productive
C
as more regularly / so more productive
D
more regularly / the more productive
解題關鍵
The + 比較級 + S+V, the + 比較級 + S+V 為固定句型，表示兩者成正比。兩項均須有 the，缺一即錯。
陷阱提示
B 缺第二個 the；C 誤用 as...so 句型；D 首項缺 The。四個選項各測一個不同的常見錯誤點。
文法補充
The + 比較級 + S+V, the + 比較級 + S+V：表示兩件事成正比變化，兩個分句均須以 the + 比較級開頭，缺少任一 the 均為語法錯誤。
單字/片語詞性音標主要意思搭配用法
the more regularly / the more productiveThe+比較級…, the+比較級…/ðə/越規律…就越有生產力✓ 標準雙重比較級結構
the more regularly / more productive缺第二個 the/ðə/第二項缺定冠詞 the錯：the more...需對稱 the more...
as more regularly / so more productiveas...so 結構/æz/as...so 為比例用法錯：as more... 不合語法
more regularly / the more productive首項缺 The/-/第一項缺定冠詞 the錯：兩項均須以 the 開頭
Q2
It was not until the auditors completed their full review ＿＿ the accounting team realized the true extent of the discrepancy.
【中譯】直到審計人員完成全面審查後，會計團隊才__意識到差異的真正程度。
A
that
B
when
C
which
D
where
解題關鍵
「It is/was not until + 時間狀語 + that + S + V」是強調「直到…才…」的固定強調句型（Cleft Sentence），that 不可替換為 when/which/where。
陷阱提示
when 是最常見陷阱——it was not until...when 在口語中偶有出現但屬非標準用法，正式寫作及考試中一律用 that，且 that 之後主要子句動詞改為肯定形式。
文法補充
強調句型 It + be + not until + 時間副詞片語/子句 + that + S + V：將「not...until」的否定語氣置於強調結構中，主要子句動詞改為肯定，表示「直到某時間點，某事才發生」。原句：The team didn't realize until the review → 強調句：It was not until the review that the team realized.
單字/片語詞性音標主要意思搭配用法
thatconj.—（It...not until...that 強調句型固定連接詞）It + be + not until + 時間狀語 + that + S + V ✓
whenconj.—（一般時間連接詞）when 用於一般時間副詞子句，不用於 it...not until 強調結構
whichpron.—（關係代名詞）此句無先行詞，不用 which 引導關係子句
whereadv.—（地點關係副詞）語義不符，此句強調時間非地點
Q3
The customer service team received a formal ＿＿ from a long-term client who was dissatisfied with the delayed shipment.
【中譯】客服團隊收到一位長期客戶的正式 __，該客戶對延誤出貨感到不滿。
A
inquiry
B
complaint
C
feedback
D
request
解題關鍵
句中關鍵詞為「dissatisfied with the delayed shipment」，客戶因不滿而提出正式文件，對應「complaint（投訴）」。complaint 特指對服務或產品不滿的正式申訴，答案為 B。
陷阱提示
inquiry 是中性的「詢問」，不含不滿情緒；feedback 泛指任何意見，語氣較中性；request 是「請求某事」，不表達不滿，均不符合「因不滿而提出」的語境。
單字/片語詞性音標主要意思搭配用法
inquiryn./ɪnˈkwaɪəri/詢問；查詢make an inquiry / respond to an inquiry
complaintn./kəmˈpleɪnt/投訴；抱怨file a complaint / handle a complaint
feedbackn./ˈfiːdbæk/意見回饋；反饋provide feedback / gather feedback
requestn./rɪˈkwest/要求；請求submit a request / honor a request
字詞家族
complaint
n. 投訴 → complain v. / complainant n.
inquiry
n. 詢問 → inquire v. / inquirer n.
feedback
n. 回饋（不可數）→ constructive/positive/negative feedback
request
n./v. 請求 → at the request of / upon request
Q4
Not only ＿＿ the project delivered ahead of schedule, but it also came in significantly under budget.
【中譯】這項專案不僅提前完工，__，更大幅低於預算完成。
A
was
B
is
C
had
D
did
解題關鍵
not only 置句首觸發倒裝，助動詞前移至主語之前；'the project was delivered'（過去被動）→ 倒裝為 'was the project delivered'；did 搭配一般動詞；had 搭配過去完成；is 時態錯誤。
陷阱提示
倒裝時助動詞的選擇取決於原句動詞形式：被動句（was/were + p.p.）→ was；一般動詞過去式 → did；過去完成 → had。本句 delivered 為被動過去分詞 → was。
文法補充
否定副詞 not only 置首倒裝：Not only + aux + S + V-p.p./V...；被動助動詞 was/were 前移；一般動詞助動詞 did 前移
單字/片語詞性音標主要意思搭配用法
wasaux/wɒz/過去被動倒裝Not only was + N + p.p.（被動） ✓
isaux/ɪz/現在式，時態不符現在式（此為過去完成之事）
hadaux/hæd/過去完成助動詞Not only had + S + p.p.（過去完成）
didaux/dɪd/過去一般動詞倒裝Not only did + S + V（一般動詞，非被動）
Q5
The new safety regulations were ------- impossible to implement within the original two-week timeline.
【中譯】新的安全法規在原定兩週時間內______不可能執行。
A
nearly
B
scarcely
C
barely
D
almost
解題關鍵
nearly impossible 是固定搭配，nearly 修飾形容詞，表示「幾乎不可能」，語氣正式，常見於商務或法律文件。scarcely 和 barely 均帶有否定語意（等同 hardly），若放於 impossible 前會造成雙重否定的邏輯問題；almost 雖語意相近，但語域較口語，且在正式書面語中 nearly 更為標準。
陷阱提示
almost 與 nearly 語意相近，但 nearly 更正式；scarcely/barely + impossible 會造成邏輯上的雙重否定，意思變成「幾乎可能」，與語境相反。
文法補充
副詞修飾形容詞時需注意語域（正式 vs. 口語）與否定語意的疊加效果
單字/片語詞性音標主要意思搭配用法
nearlyadv./ˈnɪərli/幾乎（修飾形容詞/副詞）nearly impossible
scarcelyadv./ˈskɛrsli/幾乎不（否定語氣）scarcely any time
barelyadv./ˈbɛərli/勉強、幾乎不（否定語氣）barely enough
almostadv./ˈɔːlmoʊst/幾乎（較口語）almost done
Q6
All newly hired employees are required ＿＿ an orientation session before their first day of work.
【中譯】所有新進員工在正式上班前，都必須__一場職前說明會。
A
attend
B
to attend
C
attending
D
to attending
解題關鍵
be required to-inf 是被動式的固定結構：S + be + required + to + V。主動式 require sb to-inf，被動式 sb be required to-inf。
陷阱提示
require + v-ing 用於主動語態（The task requires planning.），與本題被動語境不符。
文法補充
require 在主動語態接 to-inf（require sb to V）或 v-ing（require doing）；被動語態一律接 to-inf（be required to V）。
單字/片語詞性音標主要意思搭配用法
attendv./əˈtɛnd/出席（原形）require + 原形（✗，被動態後需接 to-inf）
to attendto-inf/tuː əˈtɛnd/出席（to 不定詞）be required to-inf ✓ 固定搭配
attendingv-ing/əˈtɛndɪŋ/出席（動名詞）require + v-ing（主動語意，不適用被動）
to attending錯誤形式—（不合文法）to + v-ing 組合不合英語語法
Q7
After three rounds of negotiations, both parties finally reached ＿＿_____ agreement that satisfied all stakeholders involved in the merger.
【中譯】經過三輪談判後，雙方終於達成了一項令所有相關利害關係人滿意的協議。
A
a
B
an
C
the
D
-
解題關鍵
agreement 以母音音素 /ə/ 開頭，初次泛指「一項協議」時須用不定冠詞，因此選 an。an 用於音標以母音開頭的名詞前（an agreement、an hour、an MBA）。
陷阱提示
選 a 是常見錯誤——規則看字母開頭（a）但應看「發音」開頭；選 the 則暗示已特定提及過，但上文並未指定某一協議；選零冠詞（-）則需複數或不可數情境。
文法補充
不定冠詞 a/an 的選擇依據是後接單字的「第一個音素」而非字母：母音音素（/æ/、/ɛ/、/ɪ/、/ɒ/、/ʌ/、/ə/ 等）前用 an，子音音素前用 a。agreement /əˈɡriːmənt/ 以 /ə/ 開頭，故用 an。
單字/片語詞性音標主要意思搭配用法
a冠詞/eɪ/不定冠詞，用於單數可數名詞初次提及或泛指a plan / a solution
an冠詞/æn/不定冠詞，用於母音音素開頭的單數可數名詞an agreement / an hour
the冠詞/ðə/ /ðiː/定冠詞，指特定或已知的事物the contract / the result
-零冠詞/—/不可數名詞泛指、複數泛指或專有名詞前不加冠詞reach consensus / gain access
Q8
The quarterly earnings report was ＿＿ disappointing that the board called an emergency meeting.
【中譯】季度盈餘報告表現__令人失望，以至於董事會召開了緊急會議。
A
so
B
such
C
too
D
very
解題關鍵
so + 形容詞/副詞 + that 引導結果子句；本句空格後為形容詞 disappointing（無名詞），故選 so 而非 such；too 接不定詞不接 that 子句；very 不引導結果子句。
陷阱提示
so 與 such 最易混淆：so + adj（後無名詞）/ such + a adj N（後有名詞）。本句 disappointing 後無名詞 → 選 so。
文法補充
so + adj/adv + that + 結果子句；such + (a/an) + adj + N + that + 結果子句
單字/片語詞性音標主要意思搭配用法
soadv/soʊ/如此地（接形容詞/副詞）so + adj + that 結果子句 ✓
suchdet/sʌtʃ/如此的（接名詞片語）such + a/an + adj + N + that
tooadv/tuː/太過（接不定詞，非 that 句）too + adj + to V
veryadv/ˈveri/非常（不引導結果子句）very serious（無 that 子句）

C

initiative 也常與 launch 搭配（如 launch a sustainability initiative），容易混淆，但 initiative 強調「政策方向或新措施」，而非以提升品牌/產品知名度為目的的宣傳運動。

endeavorn./ɪnˈdɛvər/努力；事業make every endeavor（盡一切努力）
字詞家族

Q15
＿＿ efficient the internal processes become, the faster the company can respond to market changes.
【中譯】__內部流程愈有效率，公司對市場變化的反應就愈迅速。
A
The more
B
More
C
As more
D
How more
解題關鍵

分詞構句中，完成式分詞 Having + pp 表示「先於主句動作完成的主動行為」；確認預算在前，分派任務在後，且主詞 department heads 主動確認，故選 Having confirmed。
陷阱提示
Confirmed 為過去分詞，表被動，主詞不是被確認的對象；Confirming 表同步，時間順序不符。需注意完成式分詞 Having + pp 與一般現在分詞 -ing 的差異。
文法補充
完成式分詞構句 Having + p.p. 表主動且先於主句：Having finished the report, she left the office.
單字/片語詞性音標主要意思搭配用法

Confirmedv-pp(過去分詞)/kənˈfɜːrmd/被確認過去分詞構句表被動，但主詞 department heads 是主動確認者，故不適用
