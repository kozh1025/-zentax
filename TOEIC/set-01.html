Q1
By the time the revised quotation ＿＿ sent to the client, the original offer had already expired.
【中譯】在修訂後的報價被寄送給客戶__，原始報價早已過期。
A
was
B
is
C
has been
D
was being
解題關鍵
'By the time + 過去簡單式' 描述兩個過去動作同時進行中，從句動作以過去進行式被動（was being sent）表達「正在被送出」，主句以過去完成式（had already expired）表達「已先完成」。
陷阱提示
此句易誤選 was sent（過去簡單式被動），但 by the time 引導的時間子句強調的是動作「正在進行」的背景，故須用 was being sent。
文法補充
過去進行式被動 = was/were being + pp；與過去完成式（had + pp）搭配，表示兩個過去動作的先後關係。
單字/片語詞性音標主要意思搭配用法
wasaux/wɒz/過去式 be（用於被動）主動：S + was + Ving；被動：S + was + pp
isaux/ɪz/現在式 be（用於被動）現在簡單式被動：is + pp
has beenaux/hæz biːn/現在完成式被動has been + pp（強調到現在的結果）
was beingaux/wɒz ˈbiːiŋ/過去進行式被動was being + pp（強調當時持續進行的動作）✓
Q2

解題關鍵
scheduled for + 具體時間 是 TOEIC 高頻搭配，表「正式排定於某時間點」；reserved for 表「保留給某用途或某人」；arranged for 表「為某目的而安排」；planned for 雖也接時間但較口語，正式職場公告多用 scheduled。
陷阱提示
planned for 也能接時間，但職場正式公告的標準用詞是 scheduled

B

C

解題關鍵
exacerbate = C1 正式書面「使已有的問題/情況進一步惡化」；alleviate 語意相反；aggravate 語意相近但語域較口語且含「激怒」義；provoke 偏引發新反應

Q9
The new software interface is intuitive ＿＿ for employees with no technical background to use it without training.
【中譯】新軟體介面足夠直觀，即使是沒有技術背景的員工也__無需培訓即可使用。
A
enough
B
too
C
so
D
such
解題關鍵
adj/adv + enough + for sb + to V = 足夠...使某人能夠...；intuitive enough for employees to use it 語序正確；too 搭配 to 表否定結果（太...以至於不能）；so 需搭配 that 引導結果子句；such 需後接名詞片語。
陷阱提示
enough 的位置在形容詞/副詞之後（intuitive enough），與 too 相反（too + adj）；too...to（無法做）vs. adj + enough to（足夠做）：語義相反，本句結果為「可以使用」→ enough。
文法補充
adj/adv + enough + (for sb) + to V = 足夠...（可以...）；too + adj/adv + to V = 太...（不能...）；so + adj/adv + that + S + V = 如此...以至於...
單字/片語詞性音標主要意思搭配用法
enoughadv/ɪˈnʌf/足夠（adj + enough for...to V）intuitive enough for... to use ✓
tooadv/tuː/太...（too...to，導致不能）too complicated to use（否定結果）
soadv/soʊ/如此...（so...that，需補 that）so intuitive that...（缺 that）
suchdet/sʌtʃ/如此（such + N + that）such intuitive software that...（需接名詞）

Q11
The board approved the merger with a rival firm ＿＿ financial performance had been declining for two consecutive quarters.
【中譯】董事會批准了與一家競爭公司的合併案，該公司的財務績效已連續兩季下滑。
A
whose
B
which
C
what
D
that
解題關鍵
空格後接 financial performance（名詞片語），說明空格需要所有格關係代名詞。先行詞 a rival firm 是組織（非人），但 whose 可用於代替組織的所有格（firm whose financial performance）。which 的所有格必須寫成 of which the financial performance 或 the financial performance of which；what 無先行詞；that 無所有格形式。
陷阱提示
學習者常誤認 whose 只能代替「人」，但英文中 whose 可合法用於代替組織、公司、國家等非人稱先行詞，這是高頻考點。
文法補充
關係代名詞所有格規則：whose 是唯一的關係代名詞所有格形式，可用於先行詞為人、組織、動物或事物（如 a country whose economy, a company whose revenue）。which 本身無所有格，需改用 of which。that 無所有格形式。
單字/片語詞性音標主要意思搭配用法
whoserel. pron./huːz/其…的（所有格關係代名詞）先行詞為人或組織時，所有格用 whose
whichrel. pron./wɪtʃ/（關係代名詞，代替事物）which 不能單獨作所有格（需用 of which）
whatrel. pron./wæt/（名詞性關係代名詞）what 無先行詞，不可用於限定性子句
thatrel. pron./ðæt/（關係代名詞，代替人或物）that 不能作所有格用法
Q12
By the time the project manager arrived at the office, the team ＿＿ all the preliminary reports.
【中譯】等到專案經理抵達辦公室時，團隊__已完成了所有初步報告。
A
submitted
B
was submitting
C
had submitted
D
have submitted
解題關鍵
「by the time + 過去式子句」表示「到……時為止」，主句須用過去完成式（had + p.p.）表示在參照時間點之前已完成的動作。
陷阱提示
submitted（過去簡單式）是最常見的誤選，因為外觀簡潔；但 by the time 結構要求清楚標示兩個過去動作的先後順序，必須用 had submitted 才能呈現「先完成報告，後抵達」的時序。
文法補充
by the time + S + V(past simple), S + had + p.p. ← 過去完成式公式；cf. by the time + S + V(present), S + will have + p.p. ← 未來完成式
單字/片語詞性音標主要意思搭配用法
submittedv. (past simple)/səbˈmɪtɪd/提交（過去簡單式）過去某時點的動作，無先後順序強調
was submittingv. (past continuous)/wəz səbˈmɪtɪŋ/正在提交（過去進行式）強調過去某時點正在進行，與 by the time 不搭
had submittedv. (past perfect)/hæd səbˈmɪtɪd/已提交（過去完成式）by the time + 過去式 → 主句用過去完成式 ✓
have submittedv. (present perfect)/hæv səbˈmɪtɪd/已提交（現在完成式）現在完成式不可搭配過去時間點
