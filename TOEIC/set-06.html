Q1
So impressive ＿＿ the new robotic prototypes at the expo that attendees lined up for over an hour to see the live demonstrations.
【中譯】博覽會上新款機器人原型機__如此令人印象深刻，與會者排隊超過一小時只為觀看現場示範。
A
were
B
was
C
have been
D
being
解題關鍵
「So + adj./adv. + be + S + that + S + V」是形容詞/副詞前置的結果倒裝句型。主詞 the new robotic prototypes 為複數，過去式 be 動詞用 were；that 引導結果子句（lined up = 過去式）。
陷阱提示
was 是最常見陷阱——須確認主詞 the new robotic prototypes 為複數名詞，因此 be 動詞必須用 were 而非 was。此外，being 為分詞形式，無法充當倒裝句的主要謂語動詞。
文法補充
So + adj./adv. + be + S + that 結果倒裝：將形容詞/副詞提前至句首強調，be 動詞隨之倒裝至主詞前，形成「so... be + S... that...」結構。時態與數須與主詞一致。比較正常語序：The prototypes were so impressive that...
單字/片語詞性音標主要意思搭配用法
wereaux./wɜːr/（過去式複數 be 動詞，倒裝主謂）So + adj. + were + S(複數) + that ✓（the prototypes = 複數）
wasaux./wɒz/（過去式單數 be 動詞）主詞 the new robotic prototypes 為複數，不用單數 was
have beenaux./hæv bɪn/（現在完成式）語境為過去事件（lined up），現在完成式時態不符敘事語境
beingv-ing/ˈbiːɪŋ/（現在分詞）being 不能在 So + adj. 倒裝結構中作謂語動詞
Q2
The finance team ＿＿ the quarterly report before the CEO arrived for the morning briefing.
【中譯】財務團隊在執行長來開早會之前，__ 季度報告。
A
has completed
B
had completed
C
completed
D
was completing
解題關鍵
句中有 before the CEO arrived（過去式），表示「抵達」是過去事件，而「完成報告」發生在此之前，因此需用過去完成式 had completed，表達「過去的過去」。
陷阱提示
has completed（現在完成式）用於與現在相關的情境，此句已明確為過去事件，不可用；completed（過去式）無法表達兩個過去動作的先後順序；was completing（過去進行式）暗示動作尚未完成，與 before 子句矛盾。
文法補充
過去完成式（Past Perfect）= had + p.p.，用於表示發生在另一過去動作或時間點之前已完成的動作，常與 before、by the time、when 等連接詞搭配。
單字/片語詞性音標主要意思搭配用法
has completedv. phr./hæz kəmˈpliːtɪd/已完成（現在完成式）現在完成式，強調與現在的關聯
had completedv. phr./hæd kəmˈpliːtɪd/已完成（過去完成式）過去完成式，表示先於另一過去動作完成
completedv./kəmˈpliːtɪd/完成了（簡單過去式）簡單過去式，表示過去某時完成的動作
was completingv. phr./wəz kəmˈpliːtɪŋ/正在完成（過去進行式）過去進行式，表示過去某時正在進行的動作
Q3
The revised compliance framework ＿＿ been issued last month, the department would not have faced the regulatory penalty it received in April.
【中譯】__ 修訂後的合規框架上個月已發布，該部門就不會遭受四月收到的法規處罰。
A
Were it not for the fact that ... had
B
Had the revised compliance framework
C
If the revised compliance framework had
D
Since the revised compliance framework
解題關鍵
此題考 C1 虛擬倒裝結構 were it not for，表示「要不是因為某事（反事實）」；原句為 'Were it not for the fact that the framework had been issued..., the department would not have faced...'，後接過去完成的反事實結果句。
陷阱提示
Had + S + p.p. 為已禁用文法（倒裝 If 條件句）；were it not for 與 had 倒裝形式不同，前者針對名詞短語，後者針對動作條件。
文法補充
were it not for + N/NP = if it were not for = 要不是有…（現在/未來反事實）；were it not for the fact that + clause 可延伸至子句。注意：本題結果句用 would not have + p.p.，故條件為過去反事實。
單字/片語詞性音標主要意思搭配用法
Were it not for the fact that ... hadconj./wɜːr ɪt nɒt fər/要不是（虛擬語氣，強調反事實）Were it not for X, S + would not have + p.p. ✓
Had the revised compliance frameworkconj./hæd/若已…（倒裝假設）Had + S + p.p.（倒裝If條件句，已禁用）
If the revised compliance framework hadconj./ɪf/若…（標準假設句）If + S + had + p.p.（已禁用）
Since the revised compliance frameworkconj./sɪns/由於；自從since 表原因/時間（非虛擬）
Q4
＿＿ the government's generous subsidy program, many small businesses would struggle to afford the newly __ safety standards.
【中譯】若非政府慷慨的補貼計畫，許多小型企業將難以負擔新__的安全標準。
A
Were it not for / mandated
B
If it were not for / mandating
C
Had it not been for / mandate
D
Were it not for / mandate
解題關鍵
were it not for 為 if it were not for 的正式倒裝形式，用於現在反事實虛擬；newly mandated safety standards 中 mandated 為過去分詞作後位形容詞修飾 standards。兩空缺皆需正確處理。
陷阱提示
had it not been for 語感為「若非過去某事」（過去完成虛擬），與此句表達當前假設不符；mandate 原形無法直接修飾名詞。
文法補充
were it not for + N = if it were not for + N（現在反事實虛擬倒裝）；newly mandated = recently required by official regulation（p.p. 作形容詞）
單字/片語詞性音標主要意思搭配用法
Were it not for / mandatedconj+pp/wɜːr ɪt nɒt fɔːr/若非（虛擬語氣倒裝）／強制規定的（過去分詞）were it not for + N ✓ + mandated adj.修飾standards ✓
If it were not for / mandatingconj+pres.part/ɪf ɪt wɜːr nɒt fɔːr/若非（完整虛擬）／正在強制完整形式正確但未倒裝；mandating為現在分詞不符合修飾名詞
Had it not been for / mandateconj+pp/hæd ɪt nɒt biːn fɔːr/若非（過去完成虛擬倒裝）had it not been for 用於過去事實反事實，此句為現在假設
Were it not for / mandateconj+v/wɜːr ɪt nɒt fɔːr/若非；強制規定（動詞原形）mandate 作動詞原形不能修飾 standards（需 p.p. 做後位形容詞）
字詞家族

v/n. 強制規定；授權
mandatory
adj. 強制性的
mandated
adj. 被強制規定的

n. 授命者（較少用）

Q11
------- branch manager is required to submit a monthly performance report to headquarters by the fifth of each month.
【中譯】每一位分行經理都必須在每月五日前向總部提交月度績效報告。
A
All
B
Both
C
Each
D
Every
解題關鍵
空格後 branch manager 為單數名詞，謂語動詞為 is required（單數）。Each 與 Every 均接單數可數名詞，但 Each 可置於句首後接單數名詞+單數動詞，且語意強調「逐一、個別地」，最符合此句強調個別責任的語境。All 與 Both 需接複數名詞（All managers / Both managers），與空格後 manager（無 -s）不符，可排除。Every 雖也接單數名詞，但通常不用於句首強調個別義務，Each 更為正式且精確。
陷阱提示
Every manager 語法上亦可行，但考題設計中空格後為 branch manager（單數），All 和 Both 即被排除；Each 比 Every 更強調個體責任分配，在正式商業文書中為首選。
文法補充
each/every + 單數可數名詞 + 單數動詞；all/both + 複數可數名詞 + 複數動詞。Each 強調逐一，Every 強調全部中無一例外，語意相近但搭配場合有別。
單字/片語詞性音標主要意思搭配用法
All限定詞/ɔːl/所有的（複數）All managers are
Both限定詞/boʊθ/兩者都（複數）Both managers are
Each限定詞/iːtʃ/每一個（單數）Each manager is
Every限定詞/ˈɛvri/每一個（單數）Every manager is

解題關鍵
jeopardize 強調『將某事物置於嚴重危險之中』，語氣最強烈，搭配關係、安全、交易等場景最精確；damage 較廣泛；undermine 強調逐漸削弱；compromise 除危及之意外亦常指妥協，語義較廣。
陷阱提示
damage / jeopardize / undermine / compromise 均有「危害」之意，但語氣和聚焦點不同：jeopardize 強調危險的嚴重性與即時性，最適合強調後果的警示語境。
