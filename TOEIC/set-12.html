Q1
The network operations center described the situation as ＿＿ and immediately escalated the service outage ticket to the highest priority level.

A

B
exorbitant
C
exuberant
D
exhaustive
解題關鍵
語境為「情況緊迫，立即升級到最高優先」；exigent（C1，緊迫的/要求緊急處理的）最準確描述需即時行動的緊急程度。
陷阱提示
exorbitant/exuberant/exhaustive 均以 ex- 開頭造成字形干擾，但語意均不符「緊迫情況」。
單字/片語詞性音標主要意思搭配用法

Q3
The new co-working facility, in ＿＿_ staff from all departments are encouraged to collaborate, was designed with both open and private workspaces.
【中譯】新的共用辦公空間，____所有部門的員工都被鼓勵合作，以開放式與私密式工作區並存的方式設計。
A

B

C

D

解題關鍵
「介系詞 + which」替代關係副詞 where，正式書面語常見。collaborate in a place → in which
文法補充
「介系詞 + which」正式關係代名詞子句：先行詞（地點名詞）+ 介系詞 + which + S + V，等同 where 子句，但更正式。常見：in which / at which / to which，介系詞取決於動詞搭配。
單字/片語詞性音標主要意思搭配用法
whichrel. pron./wɪtʃ/（介系詞+關係代名詞）in which = where，介系詞+關係代名詞 ✓ 正式書面語常見結構
whererel. adv./wɛr/（關係副詞）in where 語法錯誤；where 已含介系詞意義，不與 in 連用
thatconj./rel. pron./ðæt/（關係代名詞/連接詞）介系詞後不可接 that；in that 在此語境意為「因為」，語義不符
itpron./ɪt/（人稱代名詞）人稱代名詞，不構成關係子句，形成句子結構錯誤
Q4
The new safety regulations were ------- impossible to implement within the original two-week timeline.
【中譯】新的安全法規在原定兩週時間內______不可能執行。
A
nearly
B
scarcely
C
barely
D
almost
解題關鍵
nearly impossible 是固定搭配，nearly 修飾形容詞，表示「幾乎不可能」，語氣正式，常見於商務或法律文件。scarcely 和 barely 均帶有否定語意（等同 hardly），若放於 impossible 前會造成雙重否定的邏輯問題；almost 雖語意相近，但語域較口語，且在正式書面語中 nearly 更為標準。
陷阱提示
almost 與 nearly 語意相近，但 nearly 更正式；scarcely/barely + impossible 會造成邏輯上的雙重否定，意思變成「幾乎可能」，與語境相反。
文法補充
副詞修飾形容詞時需注意語域（正式 vs. 口語）與否定語意的疊加效果
單字/片語詞性音標主要意思搭配用法
nearlyadv./ˈnɪərli/幾乎（修飾形容詞/副詞）nearly impossible
scarcelyadv./ˈskɛrsli/幾乎不（否定語氣）scarcely any time
barelyadv./ˈbɛərli/勉強、幾乎不（否定語氣）barely enough
almostadv./ˈɔːlmoʊst/幾乎（較口語）almost done

Q8
By the time the project manager arrived at the office, the team ＿＿ all the preliminary reports.
【中譯】等到專案經理抵達辦公室時，團隊__已完成了所有初步報告。
A
submitted
B
was submitting
C

D
have submitted
解題關鍵
「by the time + 過去式子句」表示「到……時為止」，主句須用過去完成式（had + p.p.）表示在參照時間點之前已完成的動作。
陷阱提示
submitted（過去簡單式）是最常見的誤選，因為外觀簡潔；但 by the time 結構要求清楚標示兩個過去動作的先後順序，必須用 had submitted 才能呈現「先完成報告，後抵達」的時序。
文法補充
by the time + S + V(past simple), S + had + p.p. ← 過去完成式公式；cf. by the time + S + V(present), S + will have + p.p. ← 未來完成式
單字/片語詞性音標主要意思搭配用法
submittedv. (past simple)/səbˈmɪtɪd/提交（過去簡單式）過去某時點的動作，無先後順序強調
was submittingv. (past continuous)/wəz səbˈmɪtɪŋ/正在提交（過去進行式）強調過去某時點正在進行，與 by the time 不搭
had submittedv. (past perfect)/hæd səbˈmɪtɪd/已提交（過去完成式）by the time + 過去式 → 主句用過去完成式 ✓
have submittedv. (present perfect)/hæv səbˈmɪtɪd/已提交（現在完成式）現在完成式不可搭配過去時間點

Q10
The new software interface is intuitive ＿＿ for employees with no technical background to use it without training.
【中譯】新軟體介面足夠直觀，即使是沒有技術背景的員工也__無需培訓即可使用。
A
enough
B
too
C

D
such
解題關鍵
adj/adv + enough + for sb + to V = 足夠...使某人能夠...；intuitive enough for employees to use it 語序正確；too 搭配 to 表否定結果（太...以至於不能）；so 需搭配 that 引導結果子句；such 需後接名詞片語。
陷阱提示
enough 的位置在形容詞/副詞之後（intuitive enough），與 too 相反（too + adj）；too...to（無法做）vs. adj + enough to（足夠做）：語義相反，本句結果為「可以使用」→ enough。
文法補充
adj/adv + enough + (for sb) + to V = 足夠...（可以...）；too + adj/adv + to V = 太...（不能...）；so + adj/adv + that + S + V = 如此...以至於...
單字/片語詞性音標主要意思搭配用法
enoughadv/ɪˈnʌf/足夠（adj + enough for...to V）intuitive enough for... to use ✓
tooadv/tuː/太...（too...to，導致不能）too complicated to use（否定結果）
soadv/soʊ/如此...（so...that，需補 that）so intuitive that...（缺 that）
suchdet/sʌtʃ/如此（such + N + that）such intuitive software that...（需接名詞）

Q13
A number of employees ------- expressed interest in joining the overseas training program.

A

解題關鍵
「a number of + 複數名詞」作主詞時，動詞用複數（have）。相反，「the number of + 複數名詞」作主詞時，動詞用單數（is/was）。本句主詞為 a number of employees，故選 have。
陷阱提示

文法補充
a number of + 複數名詞 → 複數動詞；the number of + 複數名詞 → 單數動詞
單字/片語詞性音標主要意思搭配用法

Q14
＿＿ all the financial data carefully, the analyst submitted a detailed report to the board of directors.
【中譯】仔細__所有財務數據後，這位分析師向董事會提交了一份詳細報告。
A
After reviewed
B
Having reviewed
C
To have reviewed
D
She reviewed
解題關鍵
「省略主詞的完成式分詞構句」：Having + p.p. 表示分詞子句的動作發生在主要子句之前，且兩子句主詞相同（均為 the analyst）。此為 C1 高階文法考點。
陷阱提示
After reviewed 是常見陷阱——after 接分詞需加 being（After being reviewed）或改為完整子句（After she reviewed）。Having reviewed 直接省略主詞與 after，是最精簡正確形式。
文法補充
完成式分詞構句（Perfect Participial Phrase）：Having + p.p. + 受詞/補語, 主要子句。用於表達主要子句動作發生前已完成的先行動作，兩子句主詞須一致。
單字/片語詞性音標主要意思搭配用法
